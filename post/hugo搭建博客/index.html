Hugo搭建博客

Feb 11, 2022
1 min read

#Mac用户

brew install hugo

#创建新网站

hugo new site AlienchangSite

#打开主题网站https://suixinblog.cn/，选一个主题

cd AlienchangSite/themes
git clone https://github.com/vaga/hugo-theme-m10c.git

#新建文章，mysite/content

hugo new post/first.md

#本地预览

hugo server -t hugo-theme-m10c -D
http://localhost:1313/
